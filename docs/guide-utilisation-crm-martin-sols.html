1. Demarrage rapide

Le CRM sert a centraliser les operations quotidiennes Martin Sols : savoir ce qui est reserve, qui est absent, quelle caisse est controlee, quels cheques sont prets et quels documents sont a disposition.

A retenir Commence toujours par verifier le site selectionne en haut de page. Les donnees affichees dependent du site actif.
Droits Les menus visibles dependent de ton profil. Si un module manque, il faut demander une verification des droits.

Points forts des modules

Chaque module a ete pense pour eviter les fichiers disperses, les informations perdues et les controles manuels trop longs. Le CRM donne une lecture commune a toute l’equipe, tout en gardant les donnees separees par site.

2. Connexion et navigation

Se connecter

  1. Ouvre l’adresse du CRM dans le navigateur ou l’application installee.
  2. Saisis ton adresse email et ton mot de passe.
  3. Coche Rester connecte uniquement sur un appareil personnel ou professionnel securise.
  4. Clique sur Se connecter. Le CRM ouvre ensuite la page d’accueil.
Bon reflexe En cas de page blanche apres un rafraichissement, reviens sur l’adresse principale du CRM puis reconnecte-toi si necessaire.

Le menu de gauche

  • Tableau de bord doit rester le premier lien pour revenir a l’accueil.
  • Applications CRM contient les modules operationnels : reservations, location materiel, rapport de visite, ROMUS.
  • Comptabilite regroupe Controle caisse, Demandes d'acompte, Remise de cheques et Addvance.
  • Documents regroupe Promo, Fiches techniques et Procedures.
  • Administration est reserve aux profils autorises.

La barre du haut

  • Site : change le site actif, par exemple Palissy ou Pastel.
  • Cloche : affiche les notifications lorsque le CRM en propose.
  • Profil : donne acces aux parametres du compte et a la deconnexion.

3. Tableau de bord

Le tableau de bord sert de point de depart. Il donne une vue rapide sur les priorites : reservations du jour, disponibilites, conges, alertes et raccourcis vers les modules importants.

  1. Verifie le site actif en haut de page.
  2. Lis les cartes de synthese pour identifier les actions urgentes.
  3. Utilise les raccourcis pour aller directement vers location materiel, reservations, conges ou caisse.
  4. Reviens ici quand tu changes de mission ou de site.

4. Reservations vehicules

Le module reservations vehicules fonctionne avec une logique simple : choisir le vehicule, consulter le planning, puis creer ou modifier une reservation selon les droits.

Creer une reservation

  1. Ouvre Reservations vehicules dans le menu.
  2. Choisis le vehicule concerne.
  3. Selectionne la date dans le planning.
  4. Choisis le creneau disponible puis renseigne les informations demandees.
  5. Valide. La reservation apparait immediatement dans le planning.

Lire le planning

  • La vue Mois sert a voir l’occupation globale.
  • La vue Jour sert a reserver rapidement un creneau precis.
  • Un clic sur une reservation existante ouvre les informations detaillees dans une fenetre compacte.
  • Si le vehicule est deja reserve, le CRM bloque les creneaux incompatibles.

5. Location materiel

La location materiel reprend le meme principe que les reservations vehicules, mais avec des cartes visuelles pour identifier rapidement la machine.

Choisir un materiel

  1. Ouvre Location materiel.
  2. Filtre par categorie si beaucoup de materiel est affiche.
  3. Clique sur la carte de la machine voulue.
  4. Le planning de cette machine s’affiche sous les cartes.
  5. Choisis le jour et le creneau disponible.

Creneaux disponibles

  • Matin : reservation sur la demi-journee du matin.
  • Apres-midi : reservation sur la demi-journee de l’apres-midi.
  • Journee : reservation de la journee complete.
  • Certains materiels peuvent etre configures uniquement a la journee.
Lecture rapide Le point de disponibilite sur la carte donne l’etat principal ; les details restent accessibles depuis la popup de reservation.

6. Conges

Le module conges remplace le suivi Excel. Il utilise les utilisateurs existants du CRM et les rattachements au site selectionne, sans creer des salaries specifiques dans le module.

Comprendre le calendrier

  • Chaque utilisateur possede une couleur dans la legende.
  • Les lignes ou points colores representent les absences visibles sur le mois.
  • Un clic sur une date permet de consulter les absents du jour.
  • La vue jour sert a lire clairement les absences quand plusieurs salaries sont concernes.

Ajouter ou modifier un conge

  1. Choisis le bon site dans la barre du haut.
  2. Clique sur la date concernee ou sur le bouton d’ajout si ton profil l’autorise.
  3. Selectionne l’utilisateur CRM existant.
  4. Renseigne le type de conge, la date de debut, la date de fin et la demi-journee si necessaire.
  5. Valide. Le CRM signale les conflits si une absence incompatible existe deja.

7. Rapport de visite commerciaux

Le module Rapport de visite sert a organiser le travail commercial : preparer les tournees, planifier les visites clients, noter les objectifs et conserver un compte rendu exploitable par site. Il remplace les notes dispersees et donne une vision claire de ce qui a ete fait et de ce qui reste a suivre.

Ce que le commercial retrouve

  • Un calendrier mensuel pour visualiser les rapports et les visites prevues.
  • Des cartes de synthese : rapports du mois, visites realisees, rapports termines et actions a suivre.
  • Des filtres par mois, representant, statut et recherche client ou ville.
  • Une fiche detaillee pour chaque rapport avec les visites, objectifs, resultats et prochaines actions.

Creer un rapport de visite

  1. Ouvre Applications CRM > Rapport de visite.
  2. Verifie le site actif en haut de page.
  3. Clique sur Nouveau rapport.
  4. Choisis la date, le representant, le statut et precise lâ€™objectif de la tournee.
  5. Ajoute les visites clients une par une avec lâ€™heure, le contact, la ville, lâ€™objectif et le resultat.
  6. Complete ensuite la synthese du rapport et les actions a suivre.

Bien utiliser le rapport

  • Prepare la tournee avant le depart pour avoir les clients et priorites sous les yeux.
  • Apres chaque visite, renseigne le resultat pendant que les informations sont fraiches.
  • Utilise le champ Prochaine action pour noter une relance, un devis ou une nouvelle visite.
  • Passe le rapport en termine quand toutes les visites et actions principales sont saisies.
  • Filtre par representant pour suivre rapidement lâ€™activite commerciale de chaque personne.
Point fort Le responsable ou la comptabilite peut relire les rapports par site et par representant, sans chercher dans des messages ou fichiers separes.

8. Controle caisse

Le controle caisse sert a reproduire la logique du fichier Excel avec une interface CRM : factures, encaissements, sorties, comptage espece, reports et ecarts.

Multi-site Pour la comptabilite, le gros avantage est le controle rapide par site : en changeant le site dans le menu du haut, elle peut verifier Palissy, Pastel ou un autre site en un clic, sans ouvrir plusieurs fichiers.

Pourquoi c’est utile

  • La comptabilite retrouve les caisses de chaque site depuis le meme endroit.
  • Les ecarts, reports, encaissements et sorties sont visibles plus vite qu’avec un fichier Excel separe.
  • Les exports PDF donnent une base propre pour controler, archiver ou transmettre.
  • Les reperes visuels rouge/vert aident a identifier rapidement ce qui est coherent ou a verifier.

Tableau de bord caisse

  • Les cartes du haut donnent les montants clefs avec icones colorees.
  • Nouvelle caisse ouvre une popup pour choisir la date, puis charge la caisse.
  • Les alertes rouge/vert aident a verifier rapidement les saisies coherentes ou a controler.
  • Le PDF de caisse reprend les informations importantes, y compris les numeros de facture.

Liste des caisses

  1. Ouvre Comptabilite > Controle caisse > Liste des caisses.
  2. Filtre par mois et annee.
  3. Utilise les icones en fin de ligne pour voir, modifier ou supprimer une caisse.
  4. Entre dans une caisse pour ajouter les factures, entrees, sorties et pieces justificatives.

Saisir une caisse

  • Ajouter une facture ouvre une popup de saisie rapide.
  • Les entrees et sorties doivent etre ajoutees via les boutons dedies pour garder la page lisible.
  • Le comptage des especes sert a comparer le reel avec le theorique.
  • Avant validation, verifie les ecarts et les reperes couleur.

9. Demandes d'acompte

Le module demandes d'acompte centralise les demandes du site et leur suivi. Il evite les demandes dispersees et permet de voir rapidement ce qui est en attente, valide ou refuse selon les droits de l'utilisateur.

Point fort La comptabilite peut consulter les demandes par site depuis le meme menu et voir rapidement ce qui doit etre valide, complete ou relance.

Creer une demande

  1. Ouvre Comptabilite > Demandes d'acompte.
  2. Clique sur Nouvelle demande d'acompte si ton profil l'autorise.
  3. Renseigne le demandeur, le client ou chantier, le document concerne, le montant et les notes utiles.
  4. Ajoute une piece jointe ou une reference lorsque c’est necessaire pour comprendre la demande.
  5. Valide. La demande apparait dans la liste du site avec son statut.

Suivre et traiter

  • Les cartes de synthese indiquent le nombre total de demandes, les demandes en attente et les montants.
  • La liste permet de retrouver rapidement le demandeur, le document, le montant et la date.
  • Les icones en fin de ligne servent a consulter, modifier, valider, remettre en attente ou supprimer selon les droits.
  • Avant validation, controle toujours le site actif et le document associe.
Bon reflexe Une demande d'acompte doit rester lisible pour la personne qui la valide : demandeur, montant, chantier/client et justificatif doivent etre explicites.

10. Remise de cheques

Le module remise de cheques sert a preparer une remise bancaire complete : creation de la remise, ajout des cheques, controle signature/destinataire, total et impression PDF.

Point fort La comptabilite gagne du temps : chaque remise est rattachee au site actif, les montants sont totalises et le PDF peut etre controle sans ressaisie.

Creer une remise

  1. Ouvre Comptabilite > Remise de cheques.
  2. Clique sur Nouvelle remise.
  3. Choisis la date et le site si la popup le demande.
  4. Ouvre la remise creee pour acceder a sa page detaillee.
  5. Ajoute les cheques depuis cette page, un par un.

Ajouter un cheque

  • Saisis le numero de facture, le nom et le montant si tu les connais.
  • Prends une photo du cheque pour conserver le justificatif.
  • La detection automatique peut aider a remplir le nom et le montant ; corrige manuellement si la lecture est mauvaise.
  • Controle les cases Signature et Destinataire avant d’envoyer la remise a la banque.
  • Clique sur la photo pour l’agrandir et verifier les informations du cheque.
Important L’OCR est une aide. Le controle final du nom, du montant, de la signature et du destinataire reste indispensable.

11. Documents

Le menu Documents rassemble les fichiers utiles au quotidien. Les categories prevues sont Promo, Fiches techniques et Procedures.

Consulter un document

  1. Ouvre le menu Documents.
  2. Choisis la categorie : Promo, Fiches techniques ou Procedures.
  3. Utilise la recherche ou le dossier pour retrouver le fichier.
  4. Clique sur le nom ou l’icone oeil pour ouvrir le document.

Ajouter ou organiser

  • Les boutons d’ajout sont visibles uniquement avec le droit de gestion.
  • Un document peut etre rattache a un site et range dans un dossier.
  • La visibilite permet de distinguer les documents publics et internes.
  • Evite de multiplier les doublons : remplace le fichier si une version plus recente existe.

12. Tapis ROMUS

Le module Tapis ROMUS integre l’outil de generation du bon de commande dans le CRM. L’objectif est d’obtenir un PDF propre et exploitable sans quitter l’environnement Martin Sols.

  1. Ouvre Applications CRM > Tapis ROMUS.
  2. Renseigne les informations client, chantier et tapis.
  3. Complete soigneusement les mesures.
  4. Verifie le recapitulatif avant generation.
  5. Genere le PDF final et controle sa lisibilite avant envoi.

13. Profil et compte

La page profil garde les informations du compte coherentes avec le menu du haut : nom, email, photo de profil et informations personnelles selon les droits disponibles.

  • Ouvre le profil depuis l’avatar en haut a droite ou depuis /pages/account-settings.
  • Modifie uniquement les informations exactes et professionnelles.
  • Apres modification de l’email ou du mot de passe, reconnecte-toi si le CRM le demande.
  • Signale tout probleme de droit ou de site a un administrateur.

14. Smartphone et tablette

Le CRM est responsive : les memes modules sont disponibles sur ordinateur, tablette et smartphone. Sur petit ecran, le menu passe par le bouton hamburger pour laisser la place au contenu.

  • Utilise le selecteur de site en haut avant de saisir une operation.
  • Fais defiler la page pour atteindre les cartes ou le planning.
  • Les popups sont condensees pour permettre une saisie rapide.
  • Pour les photos de cheque, place le cheque a plat, bien eclaire, avec tout le montant visible.
